# Atividades de Faculdade - Programação Básica

Este repositório reúne exercícios desenvolvidos durante minha formação em Ciências da Computação, com foco na prática de programação em Java.

## 📚 Conteúdo

Os exercícios abordam conceitos fundamentais, como:
- Entrada de dados com Scanner
- Manipulação de strings
- Estruturas condicionais (if/else)
- Lógica de programação básica

## 🎯 Objetivo

Consolidar os fundamentos da programação e desenvolver raciocínio lógico por meio da resolução de problemas práticos.

## 🛠️ Tecnologias utilizadas

- Java

## 📁 Organização

Cada arquivo representa um exercício independente, contendo sua própria lógica e objetivo.

## 📌 Observação

Os códigos presentes neste repositório são voltados para aprendizado e representam minha evolução inicial na programação.
